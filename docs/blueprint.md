# **App Name**: Asraya Society Hub

## Core Features:

- Admin Dashboard: Provides a high-level overview of society metrics, including total apartments, resident count, open complaints, and parking availability. Features quick looks at recent complaints and a notice board.
- Resident Chat with AI Assistance: Secure messaging interface for admins to communicate with residents. Tenants can interact with an AI chatbot that leverages a tool to provide automated assistance about society rules, services, and events.
- Tenant Complaint Management: Tenants can file new complaints regarding issues (e.g., plumbing, electrical), describe the problem in detail, and track the status of their past and present complaints.
- Voting Platform: Platform for admins to create and manage polls on community decisions, and for residents to participate in community polls.
- Marketplace: Community-driven space to buy, sell, or rent items with other residents, overseen by admins.
- Emergency Alert System: Critical feature to broadcast urgent alerts (e.g., fire, security threats, urgent maintenance) to all residents instantly.
- Owner/Tenant Details: Allow an owner of a property to store and retrieve basic details of a tenant.

## Style Guidelines:

- Primary color: Soft blue (#64B5F6) to evoke a sense of calm and community.
- Background color: Light grey (#F0F4F8), creating a clean and modern interface.
- Accent color: Warm orange (#FFB74D) to highlight important actions and notifications.
- Body font: 'PT Sans' (sans-serif) for readability and a modern feel. Headline Font: 'Space Grotesk' (sans-serif), for a computerized and clean title aesthetic.
- Note: currently only Google Fonts are supported.
- Use simple, clear icons to represent different sections and actions.
- Card-based layout to present information in a clear and organized manner. Use of tables with sorting and searching capabilities for data-heavy pages.
